Script to download and format Flick pictures taken within regional natural parks (PNR) in France. 
The script runs with R and ```python3```, on a Linux machine. 
Make sure to install R package ```targets``` and python modules ```pyshp``` and ```shapely``` before running the script. 
Once packages are installed, open R from the main directory and run ```targets::tar_make()```.
